First run command - npm install
To the the server - npm run dev

Park a car endPoint - http://localhost:5000/CarNo/carNo 
example - http://localhost:5000/CarNo/325

Unpark the car endpoint - http://localhost:5000/Slno/SlotNo
example - http://localhost:5000/Slno/0

Get the car/slot infomation - http://localhost:5000/info/{Car No or Slot No}
http://localhost:5000/info/0  - Slot No
http://localhost:5000/info/123 - Car No
